## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

### 스케줄러 실행 (백그라운드 서비스)
```
bash# 스케줄러만 실행 (크론 작업들이 자동으로 실행됨)
npm run start:scheduler

# 또는 PM2로 데몬 실행
pm2 start dist/scheduler.js --name "batch-scheduler"
```

### 명령어로 배치 실행 (원하는 시점에 수동 실행)
```
# 일일 배치 실행
npm run batch:daily

# 주간 배치를 특정 날짜로 실행
npm run batch:weekly -- --date 2025-01-01

# 월간 배치를 드라이런으로 테스트
node dist/cli.js batch --type monthly --dry-run

# 도움말 보기
node dist/cli.js batch --help
```


```
# 스케줄러 실행
pm2 start dist/scheduler.js --name "batch-scheduler"

# API 서버 실행
pm2 start dist/main.js --name "batch-api"

# 상태 확인
pm2 status
pm2 logs batch-scheduler
```
